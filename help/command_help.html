Different Console Commands

The following commands are available in the sIRC input-field.

  • /connect "serverurl"
    • Connects you to the server specified.
  • /join #"channelname" or /j #"channelname"
    • Joins the channel specified
  • /disconnect ["quitMsg"] or /quit ["quitMsg"]
    • Disconnects you from the server. Sends quit message "quitMsg" if given.
  • /part ["partMsg"]
    • Leaves the current channel. Sends part message "partMsg" if given.
  • /nick "nickname"
    • Changes your nickname
  • /msg "user"
    • Send a private message to "user"
  • /topic "topic"
    • Sets the topic for the channel active, given that you have the permissions to do so.
  • /me "actionMessage"
    • Sends an action message
  • /wc
    • Close the current window.
  • /away ["awayMsg"]
    • Sets you away if "awayMsg" is provided, otherwise away is unset.
