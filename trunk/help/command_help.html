Different Console Commands

The following commands are available in the sIRC input-field.
  • /connect "serverurl"
    • Connects you to the server specified.
  • /join #"channelname" or /j #"channelname"
    • Joins the channel specified
  • /disconnect or /quit
    • Disconnects you from the server.
  • /nick "nickname"
    • Changes your nickname
  • /msg "user"
    • Send a private message to "user"
  • /me "actionMessage"
    • Sends an action message
  • /wc
    • Close the current window.
